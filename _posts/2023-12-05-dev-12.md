# 다양한 JOINS
- JOIN: 두 개 이상의 테이블을 특정 key를 기준으로 결합하는 것
- 테이블을 따로 두는 이유: 변경 사항이 있을때 관리 용이, 데이터 중복 최소화
```SQL
SELECT products.*, managers.name as manager_name
FROM products INNER JOIN managers on products.category = managers.managing
```
