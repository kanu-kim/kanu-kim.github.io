# 데이터의 종류
> 엑셀 기준으로 데이터 종류를 구분

- 숫자 데이터
    - 숫자로 이루어져 있는 데이터
    - 사칙연산 및 화폐, 소수점, %, 지수 등

- 문자 데이터
    - 핸드폰 번호, 숫자, 주민등록번호, 기호, 명칭
    - 문자와 숫자가 혼합된 데이터
    - 특정 함수 사용 불가
    - 문자열 함수 사용 가능

- 날짜 데이터
    - 년, 월, 일 등 날짜를 표시
    - 하이픈이나 슬래시 사용
    - 날짜 계산 가능
    - 단위가 중요함(일/시간/분)
    - 특정 함수 사용 가능

- 기타
    - 논리 데이터
    - 시간 데이터
    - 수식 데이터
